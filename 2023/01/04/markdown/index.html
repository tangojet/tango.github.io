About markdown

Veröffentlicht am 2023-01-04 in test

一、markdown是什么

it’s all about the content, about what you want to say – not about fancy formatting.

二、基础语法

1）标题

  • 一级标题：# + 空格**+**文本
  • ##二级
  • ###以此类推

2）文本

    • 斜体 *
  • ** 粗体 **
  • *** 粗斜体 ***

3）列表

  • 无序（*/+/- + 空格）
  • 有序（数字**+** . + 空格）
  • 控制层级：前面**+**tab

4）超链接

  • [描述]（地址）

    唐哥笔记

  • [描述][id]

    [id]：地址 “描述”

    唐哥笔记

  • <地址>

    https://tangojet.github.io/``

5）锚点

  • 标题后 +

    [描述] #id

    跳转到[开头]

）脚注

  • [描述]（脚注解释“脚注名字”)
  • [文本]

5）引用

  • > + 文本
  • 空格 + 文本

    这是引用的例子

  • 嵌套：多一个>则加一层

    1

    2

    3

6）分割线

  • 空行 + — / ***

7）删除线

  • 文本

    文本

8）下划线

  • 文本

9）表格

*
:-：左对齐
-:：右对齐
:-:：居中对齐

| 姓名 | 年龄 |
| ：–： |：–：|

10）图片

  • [图片描述] (图片地址)
  • 支持 jpg、png、gif、svg

美丽花儿：

# test
